01Checklist
What We Need From You
A handful of accounts, access grants and inputs — everything else (all the software) we set up and run for you. Tick each box as you send it over. Sections 1–4 are all we need to start.

Your keys stay safe: every API key and password you send is stored in an encrypted key vault (Supabase Vault) the moment it reaches us — never in plain text, and never in our logs.

1. ACCOUNTS & API KEYS
Create each under your own billing, then send us the key.
serper.dev API key — Google Search + Maps data that powers rankings and local results.
Anthropic (Claude) API key — the AI that reads the data and writes the reports.
Resend account — sends the client report emails.
2. HOSTING, DOMAIN & DNS
Your server and web address, in your name.
A server (VPS) — a Hetzner CPX31 (~$16/mo) or similar; SSH/root access, or approve us to rent one.
A domain name — e.g. aios.yourbrand.com — for the app and the client portal.
DNS / registrar access — login or permission to add records, so we can point subdomains and set up email.
3. GOOGLE SEARCH & ANALYTICS
We send one service-account email; you just add it inside Google. No passwords shared.
Google Search Console — add our service-account email as an Owner on each site.
Google Analytics 4 — add it as a Viewer, and send the numeric Property ID.
4. GOOGLE BUSINESS (GMB)
For auto-posts, review replies & Maps rank. Uses Manager access.
A verified Business Profile — verified and active (Google needs ~60 days of history).
Manager access — add the Google account we give you as a Manager.
Business details — business name, address, and location / Store ID if you have one.
5. YOUR CLIENTS & THEIR SEO
The starting set of 15 clients — a simple spreadsheet is fine.
Website address — for each client.
Target keywords — per client — plus the city / region / country to track.
Main competitors — per client (2–5 websites), if you know them.
Existing data to import — rank, backlink or audit exports (CSV / Excel). Optional.
6. BRANDING & BUSINESS
Used on the reports and the client portal.
Logo & brand colors — and fonts, if you have specific ones.
Company name & footer — contact details and report-footer text.
Team login emails — who should sign in to the portal.
Legal / company details — for invoices and the privacy & terms pages.

A server (VPS)2 Ways
Option A

Already have a server? Send us its IP address and an SSH login (root or sudo), or add the SSH key we give you.

Option B

Need one? Create a Hetzner Cloud account, add billing, make a Project, and invite our email — we set up the server (CPX31, ~$16/mo).

DNS / registrar access2 Ways
Option A

Easiest: give us login access to your registrar or DNS provider (e.g. Cloudflare), or add our email as a user — we add every record.

Option B

Prefer to do it yourself? We send a short list; for each record open DNS settings → Add record, paste the Type / Name / Value, and Save.
