Cross-system reuse — why the second build is a port, not a rebuild

What Haseeb inherits from Danyal

Frozen Pydantic contracts, the auth + FORCE-RLS pattern + CI gate, the Docker/Caddy deploy skeleton, the single ai/claude.py wrapper, and the house PDF/report stack. Built once on Danyal, ported to the second repo.

What's genuinely net-new for Haseeb

The dual-mode data layer with no Danyal equivalent: DataForSEO Standard, GSC/GA4 service-account pulls, the CSV/XLSX normalization pipeline, and the 15-client agency ops. That delta is the real Haseeb work — the striped P1 bars.

Capacity, risk & the cut-list
The capacity reality

Two full v1 platforms, 3 developers, weekdays only — 12 working days to Danyal, 15 to Haseeb, both off the same team. And Danyal's v1 just grew from four modules to five: Off-page (backlinks, citations, Web 2.0) is now in scope on the same 12-day window. That is roughly one senior FTE-month of effective capacity against what the standalone plans each scope at 3–5 weeks. It only closes because the second build ports the first and Off-page reuses the audit's DataForSEO client. Treat Danyal as the reference implementation, its P0 (Audit · Content · Portal · Radar) as mandatory core, and Off-page's Web 2.0 as the first thing the cut-list protects.

Cut-list — if behind, drop in this order
  • Off-page Web 2.0 first (Danyal): the WP.com/Blogger/Tumblr publishing pipeline defers to August; keep backlink/citation monitoring only.
  • Haseeb P1 tier: Keyword Research → Task board → Backlinks → Technical Audit move to August wholesale.
  • GA4 — keep GSC (the SEO signal); traffic becomes a manual screenshot in v1.
  • Scheduled delivery → on-demand generate + manual send.
  • Danyal Policy Radar → ship baseline recommendations only; defer change-detection automation.
  • Manager portal (Danyal) folds into Admin; multi-user roles → single shared org for July.

The rule: the core three (Haseeb) and the core four (Danyal: Audit · Content · Portal · Radar) carry ship. Off-page and everything else is negotiable before either deadline slips.

The one line to remember
Free proves value, Paid runs the platform. Danyal is the reference build; Haseeb is the port. Ship the core, protect Adan's critical path, cut Web 2.0 first.
